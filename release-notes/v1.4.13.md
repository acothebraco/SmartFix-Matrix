# DIY LED Matrix v1.4.13

## Project rename to DIY LED Matrix

This release applies the new DIY LED Matrix branding across the firmware, web configuration page, web flasher, GitHub release workflow and documentation.

### Changed

- Project name changed to DIY LED Matrix.
- Access Point SSID changed to `DIY-LED-Matrix`.
- mDNS hostname changed to `http://diyledmatrix.local/`.
- OTA binary name changed to `DIY-LED-Matrix-ota.bin`.
- USB/web flasher binary name changed to `DIY-LED-Matrix-usb.bin`.
- GitHub OTA URLs now point to `acothebraco/DIY-LED-Matrix`.
- Web configuration header, footer, preview defaults and OTA messages now use the new DIY LED Matrix branding.
- Default logo text changed to `DIY LED Matrix`.
- Default scrolling text changed to a neutral DIY LED Matrix message.

### Notes

- Because the Preferences namespace was renamed, saved device settings may reset once after flashing this version.
- After renaming the GitHub repository, update the local Git remote URL to the new repository URL.

### Firmware Assets

- DIY-LED-Matrix-ota.bin — app-only OTA update
- DIY-LED-Matrix-usb.bin — full USB/web flasher image
