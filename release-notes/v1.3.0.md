# DIY LED Matrix v1.3.0

## Text Effects and Compact Web GUI

This release adds new animation effects for the logo/header and the scrolling text. The web configuration page was reorganized with collapsible dropdown sections to keep the page compact.

### Added

- New scrolling text effects
  - Rainbow
  - Wave
  - Sparkle
  - Comet Trail
  - Flash
- New logo/header effects
  - Wave
  - Bounce
  - Glitch
  - Scanline
- Saved scrolling text effect setting in ESP32 Preferences
- Text effect status display in the web GUI
- Collapsible dropdown sections in the web GUI

### Improved

- Web configuration page is shorter and easier to navigate
- Logo and text settings are grouped by function
- Existing logo color menu is preserved
- Existing WiFi scan, mDNS and OTA functions are preserved

### Notes

- Firmware version is now v1.3.0.
- The DIY-LED-Matrix access point remains active.
- After flashing through the browser web flasher, press the RST button once if the ESP32-S3 does not restart automatically.

### Firmware Assets

- DIY-LED-Matrix-ota.bin — app-only OTA update
- DIY-LED-Matrix-usb.bin — full USB/web flasher image
