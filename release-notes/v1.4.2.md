# DIY LED Matrix v1.4.2

## Web GUI Preview and OTA Cleanup

This release cleans up the OTA section in the web configuration page and adds an animated LED matrix preview.

### Added

- Animated 64×32 LED matrix preview in the web GUI
- Live preview for scrolling text while typing
- Live preview for logo/header text while typing
- Preview support for the current text color, logo color, scroll speed and selected effects
- Preview support for the two-way slide effect

### Changed

- Firmware version is now v1.4.2
- Removed user-configurable OTA URL field from the web GUI
- Removed OTA URL save button from the web GUI
- Removed manual GitHub update check/start buttons from the web GUI
- OTA GitHub URL is now handled internally by the firmware
- OTA section now focuses on automatic GitHub update status and manual OTA BIN upload

### Notes

- Automatic GitHub update checks still run when Home WiFi is connected.
- Manual OTA BIN upload remains available in the web GUI.
- Use only `DIY-LED-Matrix-ota.bin` for manual OTA upload.
- Do not upload the full USB/web flasher BIN through the OTA upload field.

### Firmware Assets

- DIY-LED-Matrix-ota.bin — app-only OTA update
- DIY-LED-Matrix-usb.bin — full USB/web flasher image
