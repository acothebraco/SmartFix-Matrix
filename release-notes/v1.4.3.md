# DIY LED Matrix v1.4.3

## Improved LED Matrix Preview

This release improves the animated web GUI preview so the browser simulation is closer to the real 64x32 HUB75 matrix.

### Added

- Always-open LED Matrix preview panel in the web configuration UI
- Realistic LED dot rendering in the preview
- Simulated inactive and active LED pixels instead of a flat canvas text preview
- Animated Logo Text simulation in the preview
- Preview support for current Logo Text effects:
  - Static
  - Typewriter
  - Fade
  - Slide
  - Shimmer
  - Sparkle
  - Pulse
  - Wave
  - Bounce
  - Glitch
  - Scanline
  - Two-way slide
- Preview support for current scrolling text effects, including two-way slide
- Live preview updates while typing in the Logo Text and Scrolling Text fields

### Improved

- The preview can no longer be collapsed accidentally.
- The Logo Text preview now follows the selected Logo effect instead of staying static.
- The scrolling text two-way slide preview now behaves like the firmware animation, moving across the matrix and returning.
- Preview uses the current shared speed setting for Logo Text and scrolling text.

### Notes

- Firmware version is now v1.4.3.
- The browser preview is intended as a close visual simulation of the 64x32 matrix. The real LED matrix remains the final reference.

### Firmware Assets

- DIY-LED-Matrix-ota.bin — app-only OTA update
- DIY-LED-Matrix-usb.bin — full USB/web flasher image
