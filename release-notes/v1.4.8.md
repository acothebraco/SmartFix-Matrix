# DIY LED Matrix v1.4.8

## Dual Panel / 128x32 Display Support

This release adds support for two 64x32 HUB75 LED matrix panels connected side-by-side as one wide 128x32 display.

### Added

- Firmware version updated to v1.4.8
- HUB75 chain initialized for up to 2 side-by-side 64x32 panels
- New runtime display layout setting in the Web Config:
  - 1 Panel = 64x32
  - 2 Panels = 128x32
- Panel layout setting is saved in ESP32 Preferences/NVS
- Web Config status card now shows the active display layout
- LED Matrix Preview automatically adapts to 64x32 or 128x32
- Logo, header, scrolling text and effects use the selected display width

### Improved

- DIY LED Matrix header/logo is centered on the active display width
- Scrolling text uses the full selected width
- Dual-slide scroll effect now uses the full selected width
- Random FX and static logo frame use the active display width
- The second panel is cleared when switching back to 64x32 mode

### Notes

- The firmware is initialized for a maximum of two chained 64x32 panels.
- The physical connection order should be: controller to panel 1 IN, panel 1 OUT to panel 2 IN.
- For one-panel mode, content is rendered only within the first 64x32 area while the full chain is cleared.

### Firmware Assets

- DIY-LED-Matrix-ota.bin — app-only OTA update
- DIY-LED-Matrix-usb.bin — full USB/web flasher image
