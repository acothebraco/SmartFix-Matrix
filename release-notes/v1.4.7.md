# SmartFix Matrix v1.4.7

## LED Preview Logo Color Fix

This release fixes the LED matrix preview so the SmartFix logo colors match the real display more accurately.

### Fixed

- SmartFix logo preview now uses the same color logic as the real LED matrix.
- `SmartFix`, `Smart Fix`, `Smart-Fix`, and `Smart_Fix` are recognized as the SmartFix brand logo in the preview.
- Preview now respects the selected Logo Color mode for the brand logo:
  - Auto / Brand
  - Two-color by word
  - Multicolor by word
  - Single color modes
- `Smart` is rendered green and `Fix` blue when the selected logo color mode is Auto / Brand, Two-color by word, or Multicolor by word.

### Notes

- Firmware version is now v1.4.7.
- This is a web GUI preview fix only. The real LED matrix rendering was already correct.

### Firmware Assets

- SmartFix-Matrix-ota.bin — app-only OTA update
- SmartFix-Matrix-usb.bin — full USB/web flasher image
