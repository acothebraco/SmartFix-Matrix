# DIY LED Matrix v1.4.7

## LED Preview Logo Color Fix

This release fixes the LED matrix preview so the DIY LED Matrix logo colors match the real display more accurately.

### Fixed

- DIY LED Matrix logo preview now uses the same color logic as the real LED matrix.
- `DIY LED Matrix`, `DIY-LED-Matrix`, and `DIY_LED_Matrix` are recognized as the DIY LED Matrix brand logo in the preview.
- Preview now respects the selected Logo Color mode for the brand logo:
  - Auto / Brand
  - Two-color by word
  - Multicolor by word
  - Single color modes
- Words are rendered in alternating green/blue when the selected logo color mode is Auto / Brand, Two-color by word, or Multicolor by word.

### Notes

- Firmware version is now v1.4.7.
- This is a web GUI preview fix only. The real LED matrix rendering was already correct.

### Firmware Assets

- DIY-LED-Matrix-ota.bin — app-only OTA update
- DIY-LED-Matrix-usb.bin — full USB/web flasher image
