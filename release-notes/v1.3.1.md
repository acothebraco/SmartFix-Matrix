# DIY LED Matrix v1.3.1

## Web Config UI Polish

This release improves the DIY LED Matrix web configuration page and fixes the dropdown behavior after pressing buttons.

### Fixed

- Dropdown sections no longer collapse automatically after using a button or saving a setting
- Scroll position is restored after configuration actions
- WiFi scan return opens the Home WiFi section automatically

### Improved

- New web configuration design inspired by the DIY LED Matrix web flasher page
- Modern dark gradient background
- DIY badge header
- Cleaner status overview
- Better card styling and spacing
- Improved responsive mobile layout
- Styled WiFi scan page

### Technical

- Added JavaScript state persistence for open dropdown sections
- Added section IDs for direct navigation
- Stored UI state in browser localStorage/sessionStorage

### Notes

- The firmware settings and feature behavior are unchanged.
- This update focuses on the web configuration user interface.
