# SmartFix Matrix v1.4.5

## More Realistic and Smoother LED Matrix Preview

This release improves the browser LED matrix preview so it behaves closer to the real 64x32 HUB75 panel.

### Added

- Direct 5x7 pixel-font rendering in the preview instead of browser anti-aliased text
- More realistic inactive LED cups and active LED glow
- Smoother requestAnimationFrame based animation loop
- Better SmartFix brand rendering in the preview

### Improved

- Preview now draws the text into the 64x32 pixel buffer before rendering LED dots
- SmartFix preview colors match the real matrix better:
  - Smart = green
  - Fix = blue
- Scrolling text preview movement is smoother
- Logo text preview movement is smoother
- Preview now supports German umlaut pixels with the same dot logic as the firmware

### Notes

- Firmware version is now v1.4.5.
- The real LED matrix remains the final visual reference, but the web preview is now closer to the real pixel output.

### Firmware Assets

- SmartFix-Matrix-ota.bin — app-only OTA update
- SmartFix-Matrix-usb.bin — full USB/web flasher image
