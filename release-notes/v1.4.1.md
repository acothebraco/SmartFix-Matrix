# DIY LED Matrix v1.4.1

## Bidirectional Slide, Shared Logo Settings and OTA Upload

This release expands the logo and scrolling text animation controls and improves firmware update handling in the web GUI.

### Added

- New bidirectional slide effect for the scrolling text
- New bidirectional slide effect for the logo/header text
- Logo text now supports up to 160 characters
- Logo animation speed now uses the same speed setting as the scrolling text
- Logo speed controls added to the Logo/Header section
- Automatic GitHub update checks when Home WiFi is connected
- Manual GitHub update check button in the web GUI
- Latest firmware version/status display in the web GUI
- Manual OTA `.bin` upload through the web GUI
- Manual OTA flashing route for `DIY-LED-Matrix-ota.bin`

### Improved

- German UI labels now use proper umlauts, for example `Grün` and `Weiß`
- OTA default URL now points to `DIY-LED-Matrix-ota.bin`
- OTA section now clearly warns to use the OTA binary, not the USB full flash binary
- Logo and scrolling text speed behavior is more consistent

### Notes

- Firmware version is now v1.4.1.
- Manual OTA upload must use the app-only OTA file: `DIY-LED-Matrix-ota.bin`.
- Do not upload `DIY-LED-Matrix-usb.bin` through the OTA upload form.

### Firmware Assets

- DIY-LED-Matrix-ota.bin — app-only OTA update
- DIY-LED-Matrix-usb.bin — full USB/web flasher image
