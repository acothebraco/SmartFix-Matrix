# DIY LED Matrix v1.4.10

## Separate Font Size and Font Style Controls

This release adds font customization for both the logo/header text and the scrolling text.

### Added

- Separate font size setting for scrolling text
- Separate font size setting for logo/header text
- Separate font style setting for scrolling text
- Separate font style setting for logo/header text
- Four matrix-friendly font styles:
  - Classic
  - Bold
  - Wide
  - Block
- Web GUI buttons for all new font options
- Live LED matrix preview support for the selected font size and style
- Persistent storage of all font settings in ESP32 Preferences/NVS

### Improved

- Logo text and scrolling text can now be tuned independently for readability.
- Large font mode is constrained to sizes that still fit on the current 32 pixel panel height.
- Web preview now reflects font settings immediately when buttons are clicked.

### Notes

- The previous experimental 3D rotation idea is not included in this release.
- Font size is limited to Small and Large because the current 128x32 layout has only 32 pixels of height.
- The existing effects remain available and now use the chosen font style.

### Firmware Assets

- DIY-LED-Matrix-ota.bin — app-only OTA update
- DIY-LED-Matrix-usb.bin — full USB/web flasher image
