# DIY LED Matrix v1.4.9

## Separate Logo and Scroll Speeds

This release separates the speed control for the scrolling text and the logo/header text.

### Added

- Dedicated logo speed setting saved in Preferences/NVS
- New `/logo-speed` web endpoint
- Separate Web Config buttons for logo speed:
  - Slow
  - Medium
  - Fast
  - Turbo
- Preview state now includes `logoSpeed`, so the Web GUI preview follows the selected logo speed independently

### Changed

- Scrolling text speed now only changes the lower scrolling text
- Logo speed now only changes the logo/header animation
- The real matrix logo animations use `logoInterval` instead of `scrollInterval`
- On first boot after update, logo speed defaults to the currently stored scroll speed for a smooth migration

### Notes

- Firmware version is now v1.4.9.
- This release is intended as a small behavior fix after v1.4.8 dual-panel support.

### Firmware Assets

- DIY-LED-Matrix-ota.bin — app-only OTA update
- DIY-LED-Matrix-usb.bin — full USB/web flasher image
