# SmartFix Matrix v1.4.4

## GitHub OTA Install Button and Improved LED Preview

This release improves the web configuration preview and adds a direct GitHub OTA update button.

### Added

- New Web GUI button: **GitHub Update jetzt installieren** / **Install GitHub update now**
- Direct OTA flashing from the latest GitHub release asset
- Safety check before direct GitHub OTA flashing:
  - Home WiFi must be connected
  - GitHub must report a newer firmware version
  - `SmartFix-Matrix-ota.bin` must exist in the release assets
- More detailed LED matrix preview with larger 64×32 dot rendering
- Simulated LED cups, inactive LED dots, glow, grid, and front-glass effect in the preview

### Improved

- The LED preview canvas is now larger: 640×320 pixels
- The preview now represents the real LED matrix dots more accurately
- SmartFix logo preview now renders `Smart` green and `Fix` blue like the real brand logo
- Logo wave/bounce preview also uses the correct green/blue SmartFix brand colors
- OTA status messages were improved for GitHub update install attempts

### Notes

- The GitHub OTA install button only flashes when a newer release is available.
- Manual OTA BIN upload is still available.
- Always use `SmartFix-Matrix-ota.bin` for manual OTA upload, never the USB full-flash BIN.

### Firmware Assets

- `SmartFix-Matrix-ota.bin` — app-only OTA update
- `SmartFix-Matrix-usb.bin` — full USB/web flasher image
