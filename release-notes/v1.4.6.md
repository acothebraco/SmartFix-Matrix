# DIY LED Matrix v1.4.6

## Web GUI Performance Fix

This release improves the web configuration page performance after the detailed LED matrix preview introduced in v1.4.5.

### Fixed

- Reduced heavy browser rendering load from the LED matrix preview
- Prevented the preview animation from slowing down button actions
- Fixed slow reactions when pressing mode, color, effect, speed and text save buttons
- Added lightweight AJAX handling for fast web GUI controls so the full page does not reload for every setting change
- Preview is paused briefly while a setting is being saved

### Improved

- LED preview now uses a cached base matrix instead of redrawing all inactive LEDs every frame
- Active LED glow sprites are cached and reused
- Preview frame rate is limited to reduce CPU load on slower devices
- Preview stops rendering while the tab is hidden
- Small save-status toast added for fast settings changes

### Notes

- OTA upload, GitHub OTA install, WiFi scan, WiFi save, language switching and factory reset still use normal page actions for safety.
- Firmware version is now v1.4.6.

### Firmware Assets

- DIY-LED-Matrix-ota.bin — app-only OTA update
- DIY-LED-Matrix-usb.bin — full USB/web flasher image
