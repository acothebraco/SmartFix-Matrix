# DIY LED Matrix v1.4.0

## Bilingual Web UI and German Umlaut Display Support

This release adds a bilingual web configuration interface and improves Matrix text rendering for German umlauts.

### Added

- German / English language selector in the upper right corner of the web configuration page
- Persisted web UI language setting in ESP32 Preferences
- Bilingual labels for the main web configuration page
- Bilingual WiFi scan page
- UTF-8 text rendering helper for the LED matrix
- Support for German umlauts on the LED matrix:
  - Ä / Ö / Ü
  - ä / ö / ü
  - ß rendered as `ss`

### Improved

- Text width calculation now respects UTF-8 German umlauts
- Scrolling text supports umlauts in normal and animated text modes
- Logo/Header text supports umlauts in normal, wave and bounce modes
- WiFi scan signal/security labels are localized

### Notes

- Firmware version is now v1.4.0.
- The DIY-LED-Matrix access point remains active.
- The web GUI language can be switched with DE / EN in the upper right corner.

### Firmware Assets

- DIY-LED-Matrix-ota.bin — app-only OTA update
- DIY-LED-Matrix-usb.bin — full USB/web flasher image
