\# DIY LED Matrix v1.2.2



\## WiFi Scan, mDNS and Logo Color Menu



This release adds WiFi scanning, a local network hostname and a new logo color menu for the DIY LED Matrix web interface.



\### Added



\- WiFi scan button in the web GUI

\- List of available WiFi networks with SSID, signal strength, channel and security type

\- SSID selection directly from the scan result

\- mDNS hostname: http://diyledmatrix.local/

\- mDNS address display in the web GUI

\- Logo color status display

\- Logo color menu

\- Single-color logo modes: green, blue, white, yellow and red

\- Two-color word mode for logo text

\- Multi-color word mode for logo text



\### Improved



\- Home WiFi handling

\- Web GUI status overview

\- Logo/Header customization

\- Saved logo color setting in ESP32 Preferences

\- Firmware release assets for OTA and USB/web flasher



\### Notes



\- The DIY-LED-Matrix access point remains active.

\- After flashing through the browser web flasher, press the RST button once if the ESP32-S3 does not restart automatically.



\### Firmware Assets



\- DIY-LED-Matrix-ota.bin — app-only OTA update

\- DIY-LED-Matrix-usb.bin — full USB/web flasher image
