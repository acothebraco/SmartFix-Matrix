# SmartFix Matrix v1.4.11

## Brightness Slider for Web Config

This release replaces the brightness preset buttons with a compact slider in the web configuration page.

### Added

- New brightness slider in the Web Config
- Live brightness value display with raw value and percent
- Automatic saving while moving the slider
- Status card updates immediately without a full page reload

### Improved

- Brightness controls use much less space
- No more 25% / 50% / 75% / 100% button block
- Better mobile layout in the Auto Demo / Brightness section

### Firmware Assets

- SmartFix-Matrix-ota.bin — app-only OTA update
- SmartFix-Matrix-usb.bin — full USB/web flasher image
