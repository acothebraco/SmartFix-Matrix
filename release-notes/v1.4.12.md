# SmartFix Matrix v1.4.12

## Restore Font Controls and Keep Brightness Slider

This release restores the separate font size and font style controls from the previous font-control build and keeps the v1.4.11 brightness slider.

### Added

- Separate font size setting for the scrolling text
- Separate font size setting for the logo/header text
- Separate font style setting for the scrolling text
- Separate font style setting for the logo/header text
- Four matrix-friendly font styles:
  - Classic
  - Bold
  - Wide
  - Block
- Web GUI buttons for all font controls
- Live preview support for selected font size and style
- Persistent storage of font settings in ESP32 Preferences/NVS

### Kept from v1.4.11

- Brightness slider instead of brightness buttons
- Live brightness value display
- Automatic save while sliding
- AJAX save behavior without full page reload

### Improved

- Logo and scrolling text can be tuned independently for readability.
- Font settings are applied to the real matrix rendering and to the web preview.
- The release is based on the existing SmartFix Matrix 128x32 dual-panel branch.

### Notes

- The previous experimental extra font style expansion is not included.
- Only the original four font styles from the recovered ZIP are restored.
- Firmware version is now v1.4.12.

### Firmware Assets

- SmartFix-Matrix-ota.bin — app-only OTA update
- SmartFix-Matrix-usb.bin — full USB/web flasher image
