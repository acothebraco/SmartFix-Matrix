DIY LED Matrix

Flash the DIY LED Matrix firmware directly from your browser — no PlatformIO, no local toolchain.

Current firmware v__FWVER__ Build __BUILDDATE__
Use Chrome or Edge on desktop. Web Serial is required. This page must be opened via HTTPS or localhost. USB full flash · Firmware v__FWVER__
  1. Plug the Waveshare ESP32-S3 RGB Matrix board into your computer with a USB-C data cable.
  2. Click Install DIY LED Matrix and select the USB JTAG/serial or USB Serial port.
  3. For the first flash, choose Erase device + Install.
  4. After reboot, connect to the DIY-LED-Matrix WiFi access point.
  5. Open http://192.168.4.1/ to configure text, colors, effects, home WiFi and OTA.

Chrome or Edge on desktop is required because the flasher uses Web Serial. Mobile browsers usually do not support this.
